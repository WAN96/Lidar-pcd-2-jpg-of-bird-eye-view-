# Lidar-pcd-2-jpg-of-bird-eye-view-
Take bird-eye-view Jpg file from pcd file.


创建build文件夹

    mkdir build
    cd build
  
编译
    
    cmake
    make
    
运行

    ./Pcd2Bev  your file location
